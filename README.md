## Sposób interakcji

Projekt będzie miał interfejs graficzny (GUI) oparty na technologiach Django, htmx, alpine.js i tailwind.css. Użytkownicy będą mogli wchodzić w interakcje z aplikacją za pomocą przeglądarki internetowej.
